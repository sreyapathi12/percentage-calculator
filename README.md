# Percentage Calculator

A beginner-friendly Python program that calculates the percentage of marks obtained based on the total marks.

## Features
- Accepts marks obtained from the user
- Accepts total marks
- Calculates the percentage
- Displays the result clearly

## Technologies Used
- Python

## Formula Used

Percentage = (Marks Obtained / Total Marks) × 100

## Example

Input:
Marks Obtained: 450
Total Marks: 500

Output:
Percentage: 90.0%
